# Guía de Migración de Dominio (Kiplystart.com)

Tienes el código de autorización (`I"rB;-g.F!9I`), lo que significa que puedes **transferir** tu dominio fuera de Shopify. Esto es ideal para que no sigas pagando a Shopify solo por el dominio.

## Opción Recomendada: Transferir a Vercel
Como vamos a alojar tu web en Vercel, transferir el dominio ahí hace que todo sea automático.

### Pasos:
1.  Crea tu cuenta en [vercel.com](https://vercel.com) y conecta tu repositorio de GitHub (donde subiremos este código).
2.  Ve a **Domains** en tu dashboard de Vercel.
3.  Haz clic en **"Add Domain"** o **"Transfer In"**.
4.  Escribe `kiplystart.com`.
5.  Vercel detectará que está bloqueado o requiere código.
6.  Ingresa el **Código de Autorización** que tienes en la imagen: `I"rB;-g.F!9I`
7.  Paga la tarifa de transferencia (que generalmente te renueva el dominio por un año más).

---

## Opción Alternativa: Namecheap o Cloudflare
Si prefieres un registrador dedicado (a veces más barato, ~$10/año):

1.  Ve a [Namecheap.com](https://www.namecheap.com/domains/transfer/) o [Cloudflare](https://www.cloudflare.com/products/registrar/).
2.  Selecciona "Transfer Domain".
3.  Ingresa `kiplystart.com`.
4.  Pega el código de autorización: `I"rB;-g.F!9I`.
5.  Una vez transferido, tendrás que configurar los **DNS** para que apunten a Vercel (te daré esos datos cuando despliegues).

> [!IMPORTANT]
> **No pierdas ese código.** Es la llave maestra de tu dominio. Si no lo usas antes del 24 de septiembre de 2026, el dominio caducará, pero es mejor hacerlo YA para desconectarte de Shopify completamente.
