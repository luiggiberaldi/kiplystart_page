# Guía de Configuración Supabase

Para que el formulario de contacto funcione, sigue estos pasos:

## 1. Crear Proyecto
1.  Ve a [supabase.com](https://supabase.com) y crea una cuenta.
2.  Crea un nuevo proyecto llamado `KiplyStart`.

## 2. Configurar Base de Datos
1.  Ve al **SQL Editor** (barra lateral izquierda).
2.  Pega y ejecuta el siguiente código para crear la tabla de mensajes:

```sql
create table messages (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  name text,
  email text,
  message text
);
```

Ensure Row Level Security (RLS) policies allow inserts if necessary, or disable RLS for testing (not recommended for production but easiest for quick start):
```sql
alter table messages enable row level security;

create policy "Enable insert for everyone" 
on "public"."messages" 
as PERMISSIVE 
for INSERT 
to anon 
with check (true);
```

## 3. Obtener Claves API
1.  Ve a **Project Settings** (icono de engranaje) -> **API**.
2.  Copia la **Project URL**.
3.  Copia la clave **anon** / **public**.

## 4. Conectar al Proyecto
1.  Crea un archivo llamado `.env` en la raíz de tu carpeta `kiplystar pagina`.
2.  Pega las claves siguiendo el formato de `.env.example`:

```
VITE_SUPABASE_URL=tu_project_url_aqui
VITE_SUPABASE_ANON_KEY=tu_anon_key_aqui
```

¡Listo! Reinicia la terminal (`npm run dev`) y el formulario guardará los mensajes en tu base de datos.
